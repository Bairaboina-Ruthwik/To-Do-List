# 🧠 To-Do List Manager (Python CLI)

A simple and beginner-friendly **command-line To-Do List application** built using Python.  
This project helps you manage daily tasks efficiently while also demonstrating core Python concepts like lists, file handling, loops, and exception handling.

---

## 🚀 Features

- ✅ Add new tasks with timestamp  
- 📋 View all tasks  
- ❌ Delete tasks  
- ✔️ Mark tasks as DONE  
- 💾 Automatically saves tasks to a file (`task.txt`)  

---

## 🛠️ Tech Used

- Python (Core Concepts)
- File Handling
- Date & Time (`datetime` module)

---

## 📂 Project Structure

To-Do-List/
│── todo.py       # Main program
│── task.txt      # Stores tasks (auto-created)
│── README.md     # Project documentation


---

## ▶️ How to Run

```bash
git clone https://github.com/your-username/To-Do-List.git
cd To-Do-List

```
